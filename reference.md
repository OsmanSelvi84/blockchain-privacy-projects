# Reference Implementation

Reference Project: DIDKit

Repository:
https://github.com/spruceid/didkit

Purpose:
DIDKit is an open-source toolkit for working with Decentralized Identifiers (DIDs) and Verifiable Credentials.

Reason for Selection:
This project demonstrates practical DID creation, resolution, and identity management concepts similar to the objectives of this assignment.

Comparison:
The original implementation in this repository provides a simplified DID registry supporting:

- DID Creation
- DID Update
- DID Revocation
- DID Resolution

The reference implementation provides these concepts in a production-grade ecosystem.
